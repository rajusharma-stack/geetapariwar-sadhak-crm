Geeta Pariwar Nepal Sadhak CRM
================================

HOW TO INSTALL & RUN ON ANY PC
-------------------------------
1. Install Python 3.13+ from https://www.python.org/downloads/
   → IMPORTANT: Check "Add Python to PATH" during installation

2. Double-click "setup.bat" (one-time setup)
   - Creates virtual environment
   - Installs required packages
   - Initializes database
   - Creates Desktop shortcut

3. After setup, either:
   - Double-click "run.bat" to launch
   - Or use the Desktop shortcut "GeetaPariwarCRM"
